# Trabajo en Clase Student Web
## Clase: Arquitectura de Software 

Este proyecto consiste en el desarrollo de una aplicación web utilizando **JavaServer Pages (JSP)** desplegada sobre **Apache Tomcat**, cuyo propósito es capturar y procesar información de un estudiante. La aplicación permite ingresar el nombre y la fecha de nacimiento, calcular la edad automáticamente, almacenar la información en sesión y generar una respuesta dinámica con un saludo dependiendo de la hora del día.

---

## Objetivo
Implementar una aplicación web básica basada en JSP que permita:

- Comprender el funcionamiento de JSP dentro de un servidor web.
- Utilizar JavaBeans para encapsular lógica de negocio.
- Manejar datos en sesión.
- Generar contenido dinámico.
- Aplicar validaciones y control de flujo con JSTL.

--- 

## Tecnologias Utilizadas:

- Java (JDK 8+)
- JSP (JavaServer Pages)
- JavaBeans
- JSTL (JSP Standard Tag Library)
- Apache Tomcat 9
- NetBeans IDE
- HTML5
- CSS3
